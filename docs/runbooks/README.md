# Runbooks Index

`docs/runbooks/` は、定型的な運用手順の置き場です。
実務で繰り返す作業は、今後ここに寄せる前提で扱います。

## 使い方

- オペレーション担当が手順をそのまま実行できる粒度で読む
- 仕様ではなく、前提条件・手順・確認項目を主に書く
- ルール変更があったら、対応する設計書や ADR と一緒に更新する
